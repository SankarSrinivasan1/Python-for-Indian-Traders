# User Guide

## Introduction

Indian Stock Screener helps traders quickly identify stocks matching technical trading strategies.

The application provides a simple dashboard where users choose a strategy and instantly receive qualifying stocks.

---

# Home Screen

The dashboard contains

- Sidebar
- Strategy Selection
- Parameters
- Results Table
- Export Button

---

# Select a Screening Strategy

Choose one of the following.

## RSI Oversold

Returns stocks where

```
RSI < 30
```

Suitable for finding possible reversal candidates.

---

## RSI Overbought

Returns stocks where

```
RSI > 70
```

Useful for identifying potentially overextended stocks.

---

## Golden Cross

Returns stocks where

```
50 EMA crosses above 200 EMA
```

Often considered a long-term bullish signal.

---

## Supertrend Buy

Returns stocks currently above the Supertrend indicator.

Useful for trend-following traders.

---

## ADX Trend

Shows stocks with strong directional movement.

Higher ADX generally indicates a stronger trend.

---

## Bollinger Breakout

Returns stocks trading above the upper Bollinger Band.

Useful for momentum traders.

---

# Running the Screener

1. Launch the application.
2. Choose a screening strategy.
3. Adjust indicator settings if required.
4. Click **Run Screener**.
5. Wait for analysis to complete.
6. View qualifying stocks.

---

# Results Table

Each row displays

- Stock Symbol
- Company Name
- Latest Price
- Indicator Values
- Screening Status

---

# Sorting Results

Click any column heading to sort.

Examples

- Highest RSI
- Lowest RSI
- Highest ADX
- Alphabetical order

---

# Export Results

Click **Export CSV**.

Results are saved for further analysis.

---

# Best Practices

- Scan after market close.
- Confirm signals using price action.
- Use multiple indicators.
- Apply proper risk management.
- Never rely on a single indicator.
